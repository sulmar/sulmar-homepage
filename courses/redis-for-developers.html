Redis 7.0 dla programistów .NET

Redis 7.0 dla programistów .NET

Redis to magazyn danych o strukturze klucza i wartości typu open source, przechowywany w pamięci, należący do kategorii baz danych NoSQL. Redis może być używany jako wysokowydajna pamięć podręczna w pamięci, broker komunikatów i silnik przesyłania strumieniowego. Dzięki temu kursowi zdobędziesz umiejętność tworzenia wydajnych aplikacji przy użyciu Redis i .NET.

Czego się nauczysz

  • Confidently build front-end apps with React and TypeScript
  • Build reusable function components
  • Style your components using vanilla CSS, CSS modules, and CSS-in-JS
  • Use VSCode shortcuts to increase your productivity
  • Deploy your React apps
  • Write clean code like a pro
